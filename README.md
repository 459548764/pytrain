# pytrain

Machine Learning library for python

## Algorithms

+ Decision Tree(ID3)
+ Gaussian NaiveBayes
+ NaiveBayes
+ KNN
+ Neural Network(FNN)
+ Logistic Regression
+ Linear Regression
+ DBSCAN
+ Apriori
+ Kmeans
+ HierarchicalClustering
+ BinarySVM
+ SVM [Not yet]
+ HMM [Not yet]
+ CRF [Not yet]

## Requirements

 - Numpy
 - Python 2 or 3

## Installation

    $ sudo pip install --upgrade pytrain
    
## Basic Usage

[See examples here](https://github.com/becxer/pytrain/tree/master/examples)

## How to contribute

    Fork this repository, and write your algorithm, pull request.
    Don't forgot proper test code in test_pytrain.
    Test code should be work successfully in below command.
    
    $ python test.py
